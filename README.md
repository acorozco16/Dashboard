# Warehouse P&L Dashboard

Multi-site 3PL financial model and scenario planner.

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign up/login
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

Your dashboard will be live at `your-project-name.vercel.app`

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
